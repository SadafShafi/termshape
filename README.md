# tremshape
Tremshape is a minimalistic Python packgage, that only prints basic 
shapes on terminal. 
It does not have any dependencies.

## Usage

* Print a square:
```python
print_square(5,5)
```
```
* * * * *
*       *
*       *
*       *
* * * * *
```

* Print a rectungle:
```python
print_rect(10,5)
```
```
* * * * * * * * * *
*                 *
*                 *
*                 *
* * * * * * * * * *
```


* Print a triangle:
```python
print_triangle(10)
```
```
*                  
* *                
*   *              
*     *            
*       *          
*         *        
*           *      
*             *    
*               *  
* * * * * * * * * *
```
